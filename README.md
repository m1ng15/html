# HTML (: 제목)
## HTML 기초 (: 부제목)
### 태그 (: 소제목)
* `title` : 제목 태그
* `meta` : 문서 정보 태크

### HTML 중요 키워드
* `markup` : HTML을 의미적으로 작성한다 == 마크업
* `Debugging` : HTML 포함 프로그래밍 언어들의 오류를 검사하여 정상코드로 만든다

## Vs Code 주요 단축키
* `Ctrl + /` -> 한 줄 주석
* `Ctrl + Alt + A` -> 블록 주석
* `Ctrl + Alt + 방향키` -> 해당 방향으로 복제
